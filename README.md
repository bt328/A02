# A02
How to use Git and Github with Visual Studio Code  
Section 1: Installations  
Download Git: https://git-scm.com/install/  
Create a Github account: https://github.com/signup  
Download Visual Studio Code: https://code.visualstudio.com/download  
Install the GitHub Pull Requests and Issues extension in VS Code: https://marketplace.visualstudio.com/items?itemName=GitHub.vscode-pull-request-github  
In VS code, click the Github Icon in the Activity Bar and sign in with your account.  
Section 2: Setting Up the Repository  
In Github, click on your profile, then repositories, and then new. After filling out some basic information, you can create a repository.  
You can clone an existing repository from Github by opening the command palette with Ctrl+Shift+P and using the Git: Clone command.  
Section 3: Making Changes, Branches, and Commits  
Open the command palette and use Git: Rename Branch to rename your branch  
Clicking the Source Control icon opens a panel to view and commit changes  
You must add a short message about your changes before you can commit them  
Open the command palette and use Git: Create Branch to create a new branch  
Merging branches can be done by clicking the three dots in the Source Control menu, then clicking Branch>Merge  
Click on the name of the branch in the bottom left corner to switch between different branches
